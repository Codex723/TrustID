# Contributing to TrustID

We welcome contributions! Please see our issue tracker for "good first issues".

## Development Setup

1. Install Node.js dependencies: `npm install`
2. Install Rust and Soroban CLI to work on contracts.
3. Run dev server: `npm run dev`

## Drips Wave

TrustID is part of the Drips Wave! We support sustainable open-source funding.

## Code of Conduct

Please be respectful and professional in all communications.
